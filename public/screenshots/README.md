# Screenshots

Add your app screenshots here following these guidelines:

## Directory Structure

```
public/screenshots/
├── iphone/
│   ├── 1.png
│   ├── 2.png
│   ├── 3.png
│   ├── 4.png
│   └── 5.png
└── ipad/
    ├── 1.png
    ├── 2.png
    └── 3.png
```

## Recommended Sizes

### iPhone Screenshots
- **Width**: 260px (or multiples like 520px, 780px)
- **Aspect Ratio**: 9:16 (portrait)
- **Example Size**: 260x462px or 390x844px
- **Format**: PNG, WEBP, or AVIF

### iPad Screenshots
- **Width**: 360px (or multiples like 720px)
- **Aspect Ratio**: 4:3 (portrait or landscape)
- **Example Size**: 360x480px or 768x1024px
- **Format**: PNG, WEBP, or AVIF

## Tips

1. Use high-quality screenshots from actual devices
2. Optimize images before adding (use tools like TinyPNG, Squoosh)
3. Consider using WEBP or AVIF for better compression
4. Keep file sizes under 200KB per image when possible
5. Use descriptive filenames if you want (update src/config/screenshots.js accordingly)

## Updating Configuration

Edit `src/config/screenshots.js` to add/remove screenshots or change paths.
